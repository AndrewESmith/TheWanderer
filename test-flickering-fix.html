Technical Details

The flickering was caused by:

  • Physics simulation running on every move, recreating the maze array
  • React re-rendering all cells when the maze reference changed
  • CSS transitions and will-change properties causing visual artifacts
  • Unnecessary re-renders when movement was blocked

The fix implements:

  • Reference-based maze change detection
  • Optimized Cell component with type-only comparison
  • Hardware-accelerated CSS without transitions
  • Early return optimization for blocked movements
